Tag h1 - h6

As tags "H" são usadas para criações de titulos destacados. A Tah h1, por sua vez cria o titulo principal e maior, as demais tags h´s, quanto maior o seu numero, menor é o tamanho do seu titulo gerado.

Tag P

A tag P, nada mais é usada se não para a criação de caixas de textos ou paragrafos

Tag Mark

Ele é geralmente usado para destacar palavras-chaves em um resumo, nomes de produtos em um comentário ou outros vãos de texto cuja a apresentação típica seria negrito

Tag small

É Usada quando há necessidade de apresentação de um texto em formato pequeno

tag i

É usada para formatar palavras em italico

Tag u

Usada para sublinhar palavras ou textos

Tag strong

Para formatar em negrito

Tag ol

Para criar uma Lista Ordenada

tag ul

Para criar uma Lista não ordenada

Tag a

Para criar um link para uma pagina externa ou interna

Tag hr

Para inserir uma linha separatoria horizontal

Tag sup

Um trecho do texto pode ser formatado como subscrito ou sobrescrito usando-se as tags sub ou sup. Essas tags são muito úteis em fórmulas químicas e em algumas equações matemáticas. É só copiar e colar o código em seu conteúdo!

Tag blockquote

Geralmente usada para inserir citações
